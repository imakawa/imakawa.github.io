仝AngualrJs々の古勣
この嫗では、仝AngualrJs々の旋喘開兵圭隈と、実廾や処尖を尖盾するための秤報を紹初します。
１−１
仝AngualrJs々とは
仝AngularJs々は、JavaScriptフレームワークには寇しい、仝データ・バインディング々や仝DI々を採喘した、仝MVC(Model・View・Controller)フレームワーク々です。
賞寄な仝AJAXアプリケーション々の開発などに適しています
JavaScriptのフレームワークである仝AngularJS々は、Googleの麼導によって、オーペンソースで開発されています。
2012定6埖13晩に仝1.0.0々が発燕され、執筆現壓のバーションは仝1.2.0々です2013定11埖
仝MITライセンス々として巷開されており、仝广恬権燕幣々と仝云許諾燕幣々を記載すれば、無償で無崙泙棒喘できます。
１−２
仝AngularJS々の蒙徴
仝AJAX々は、仝XMLHttpRequest々の掲揖豚宥佚を旋喘した室術
仝AJAX々は、仝Asynchronous Javascript + XML々の待で、仝XMLHttpRequest々の掲揖豚宥佚を旋喘した室術の総各です。
仝JSON々や仝XML々を旋喘して仝html々の匯何を書き換えることを朕議とし、仝Gmail々、仝GoogleMap々、仝Google検沫々などに聞われています。
仝褒圭鬟ータ・バインディング々を採喘
仝データ・バインディング々は、仝データ々と仝ビュー々を結栽して旋喘する室術で、參和の2種類があります
(a)仝匯圭鬟ータ・バインディング々
仝データ々の変厚に応じて仝ビュー々が変厚される。
(b)仝褒圭鬟ータ・バインディング々
仝ビュー々仝データ々のどちらかが変厚されると、もう頭圭も変厚される。
仝AngularJS々では(b)の仝褒圭鬟ータバインディング々を採喘しています。
仝DI々の採喘で仝コンポーネント晒々否叟に
仝DI々は、仝Dependency Injection々の待で仝卆贋來の廣秘々を訳します。
仝ソースコード々と仝コンポーネント々間の卆贋をなくし、翌何ファイルなどを聞うことで、それらを盾決する室術です。
この仝DI々を旋喘することで、ソースコード間の卆贋業は詰和し、仝コンポーネント晒々が簡単になります。
MVC坤▲廛蠅髻献皀妊襦后献咼ー々仝コントローラ々で蛍護
仝MVC々とは、アプリケーションを仝モデル々仝ビュー々仝コントローラ々の眉つに蛍護して、設計開発する返隈です。
モデル
アプリケーションが扱うデータや、ビジネスロジックを扱う勣殆です
データの変厚を仝ビュー々に宥岑するのも、仝モデル々の碧並になります。
ビュー
ユーザーに見せるために、仝モデル々のデータを旋喘して、仝html々を燕幣する機嬬です。
コントローラ
ユーザーがデータを秘薦すると、それに応じて処尖をするものです。々
仝コントローラ々は、仝モデル々や仝ビュー々の変厚を柵び竃すことがありますが、仝モデル々のデータを岷俊荷恬することはありません。
